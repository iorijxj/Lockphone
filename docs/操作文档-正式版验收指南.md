## 第 2 步：首次设置（向导）

手机上打开 **Lockphone**，这次它会进入设置向导：

1. **设置家长 PIN**：输入至少 4 位数字，两次一致 →「下一步」。**PIN 记在安全的地方，别忘了**（忘了也能用文档最后的救援命令解，但麻烦）
2. **勾选白名单**：勾上允许孩子用的 APP（**记得勾多邻国**），点「完成并进入锁定」
3. **【观察】**点完「完成」的瞬间，桌面有没有闪烁、短暂黑屏或卡住？如实记录（预期是直接顺畅进入锁定桌面）

## 救援命令（出问题时用，平时不要执行）

万一被锁死/程序异常/忘记 PIN，连电脑执行：

```
C:\adb\platform-tools\adb.exe shell dpm remove-active-admin com.lockphone/.admin.LockAdminReceiver
```

手机立刻恢复自由（数据无损）。之后想重新启用：重新执行激活命令即可（预检指南第 8 步那条）。
